# LogicHive Function Store MCP
**Draft-First AI-Native Function Repository.**

AIエージェント（LLM）が自ら生成したコード（関数）を保存し、世界中のエージェント間で共有・再利用するための「関数のグローバルリポジトリ」へ接続するための MCP (Model Context Protocol) サーバーです。

## 💡 解決する課題
AIエディタの普及によりコード生成のハードルは下がりましたが、以下の課題があります：
- **再利用性の欠如**: エージェントごとに都度ゼロからコードを生成するため、遅延や品質のブレが生じる。過去の成功パターンが組織的に蓄積されない。
- **共有手段の不足**: 汎用的な機能を複数のプロジェクトや他のユーザーと簡単に共有・再利用する仕組みがない。

LogicHive Function Store MCP を導入することで、あなたのAIアシスタントは「過去に保存された高品質な関数」を検索・再利用できるようになり、複雑な機能の開発を劇的に高速化します。

## 🚀 使い方 (Quick Start)

### 1. インストール
本リポジトリをクローンし、初期セットアップツールを実行します。

```bash
uv run logic-hive-setup
```

### 2. MCP クライアントへの登録
お使いのAIエディタ（Claude Desktop, Cursor, Cline 等）のMCP設定ファイル（`mcp_config.json` 等）に本サーバーを登録します。

**設定例 (Claude Desktop)**:
```json
"mcpServers": {
    "logic-hive-function-store": {
        "command": "uv",
        "args": [
            "--directory",
            "C:/path/to/function-store-mcp",
            "run",
            "mcp-server-function-store"
        ]
    }
}
```

### 3. AI への指示（ワークフロー）
AIエージェントに以下のように指示するだけで、関数の検索や保存が自動で行われます。
- 「〜の機能を実装したいので、Function Store から似た関数を探して」
- 「今作ったこの機能を汎用メソッドとして Function Store に保存して」

## ⚠️ 【重要】MVPフェーズおよびライセンスに関するお知らせ

現在、本システムは **MVP（Minimum Viable Product）フェーズ** として**完全無料**で提供されています。システムの運用上、以下の点に必ずご留意ください。

### MITライセンスの自動適用
本フェーズにおいて、Function Store を経由して保存（Push）された全てのコードや関数データは、**自動的にオープンソースの「MIT ライセンス」が適用されます。**
- 保存されたファイルは、パブリックリポジトリにて全世界に公開されます。
- 他のユーザーやあらゆるサードパーティが、あなた（のAI）が登録したコードを自由に検索・再利用・改変・商用利用できるようになります。
- **🚨 企業秘密、個人情報、非公開プロジェクトの固有ロジックなど、外部に漏洩してはならないコードは絶対に本システムを使用して保存しないでください。**
